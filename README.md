# Moqing Editor - Professional Python Code Editor

## 📋 Project Description

Moqing Editor is a professional-grade code editor built with Pygame, featuring syntax highlighting, intelligent code completion, and a smooth, modern interface. It's designed to provide a lightweight yet powerful coding experience similar to PyCharm or VS Code.
(It's actually just for fun.Not very professional)

## ✨ Key Features

### 🎨 **Syntax Highlighting**
- Support for multiple languages: Python, JavaScript, HTML, CSS, Java
- Auto-detection of programming language
- Professional color schemes for each language

### ⌨️ **Code Editing**
- Smooth cursor movement with animation
- Auto-indentation when pressing Enter
- Smart code completion (Ctrl+Space)
- Multi-line selection with mouse drag
- Copy/Cut/Paste (Ctrl+C, Ctrl+X, Ctrl+V)
- Select All (Ctrl+A)

### 🔍 **Navigation**
- Double Shift to open search
- Page Up/Down for quick scrolling
- Ctrl+Mouse Middle Click to center viewport
- Smooth viewport following

### 📁 **File Operations**
- Open files (Ctrl+O)
- Save files (Ctrl+S)
- Save As (Ctrl+Shift+S)
- Working directory management
- 
### 🎮 **Shortcuts**
| Shortcut | Action |
|----------|--------|
| Ctrl+O | Open file |
| Ctrl+S | Save file |
| Ctrl+Shift+S | Save As |
| Ctrl+C | Copy |
| Ctrl+V | Paste |
| Ctrl+X | Cut |
| Ctrl+A | Select All |
| Ctrl+Alt+L | Format code |
| Ctrl+/ | Toggle comment |
| Ctrl+Space | Show completions |
| F1 | Settings |
| Double Shift | Search |
| Page Up/Down | Scroll page |
| Ctrl+Middle Click | Center viewport |

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/zhdljc/moqing-editor.git

# Install dependencies
pip install pygame

# Run the editor
python main.py
```
